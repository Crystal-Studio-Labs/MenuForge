## 🛡️ Our Commitment

We take the security of **MenuForge** and its users very seriously. We are committed to protecting the integrity of our software and the data of those who use it. This policy outlines our procedures for reporting and handling security vulnerabilities.

<br>

## ✅ Supported Versions

We actively provide security updates for the following versions of MenuForge:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

<br>

## 🚨 Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public issue.** 

Instead, please report it through one of the following secure channels:

1.  **Email:** Contact us directly at **[connect.crystalstudio@gmail.com](mailto:connect.crystalstudio@gmail.com)**.
2.  **GitHub Private Reporting:** Use the ["Report a vulnerability"](https://github.com/Crystal-Studio-Labs/MenuForge/security/advisories/new) button on the Security tab of our repository.
